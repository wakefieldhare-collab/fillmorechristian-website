The WOFF2 files in this directory are local Latin subsets served from the
Fillmore Christian Church site so pages do not depend on Google-hosted font
CSS at runtime.

- Playfair Display: https://fonts.google.com/specimen/Playfair+Display
- Source Sans 3: https://fonts.google.com/specimen/Source+Sans+3

Both font families are distributed through Google Fonts under the SIL Open Font
License 1.1. If these files are refreshed later, keep the filenames stable or
update `css/style.css` and the migration readiness checks at the same time.
